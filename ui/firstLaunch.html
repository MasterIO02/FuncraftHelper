Bienvenue sur FuncraftHelper !

Quel launcher sera utilisé pour jouer sur Funcraft ?

AZ Launcher

Launcher Minecraft Officiel

Chrome doit être installé avant d'utiliser FuncraftHelper (vous pouvez changer son emplacement dans les paramètres).

Vous pouvez utiliser un autre navigateur basé sur Chromium (comme Edge, Vivaldi, Opera par exemple), mais si des problèmes de navigation se produisent aucun support ne sera fourni.

En utilisant FuncraftHelper, vous acceptez d'envoyer des données d'utilisation comprenant la configuration de FuncraftHelper et votre pseudo.
